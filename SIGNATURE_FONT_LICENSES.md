# PixoPDF Signature Font Licenses

To comply with privacy laws and license restrictions, PixoPDF does not fetch external handwriting files or load third-party assets from Google Fonts. 

Every handwriting option rendering typed signatures utilizes system-level CSS handwriting and cursive font fallbacks pre-licensed on the user's local operating system:

| Style ID | CSS Font Family Declaration | Licensing & Redistribution |
|---|---|---|
| Style 1 | `Brush Script MT`, `Apple Chancery`, `cursive` | Standard Pre-licensed Operating System Font |
| Style 2 | `Segoe Print`, `Comic Sans MS`, `cursive` | Standard Pre-licensed Operating System Font |
| Style 3 | `Lucida Handwriting`, `cursive` | Standard Pre-licensed Operating System Font |
| Style 4 | `cursive` | Standard Generic Browser Cursive Fallback |
