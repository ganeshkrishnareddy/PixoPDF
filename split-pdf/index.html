Frequently Asked Questions

What is Split Every Page mode?

This automatically divides every page in the PDF into separate single-page documents and compiles them locally into a download-friendly ZIP archive.

How does the range syntax work?

You can specify intervals using hyphens (e.g. 1-3) or discrete elements using commas (e.g. 1, 4, 7). You can combine them, like: 1-3, 5, 8-10.
