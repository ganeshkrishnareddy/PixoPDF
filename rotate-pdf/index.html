Frequently Asked Questions

Can I rotate only a single page of a PDF?

Yes. PixoPDF renders a visual preview grid. Clicking the rotate icon on individual page thumbnails will apply a clockwise 90-degree step to only that specific page.

Does rotating pages degrade document clarity?

No. PixoPDF updates only the viewport rotation flags within the PDF structure itself using pdf-lib. The underlying text vectors and raster images are preserved exactly with zero quality degradation.
